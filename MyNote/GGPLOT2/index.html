                                  自定义标度
                                  参数描述
                                  构造器

                                  • super: 用于构建标度的超类(父类);

                                  • name: 标度的名称, 将作为轴或图例标题;

                                  • breaks: 指定轴或图例显示主要刻度的位置;

                                  • labels: 指定轴或图例显示主要刻度的名称;

                                  • drop: 指定是否从数据移除未使用的因子水平;

                                  • na.translate: 指定离散型标度是否显示缺失值;

                                  • minor_breaks: 指定轴或图例显示次要刻度的位置;

                                  • scale_name: 指定标度发生错误时关联的标度名称;

                                  • trans: 指定用于连续型数据转换的转换器名称或转换器函数;

                                  • show.limits: 指定是否将标度的上限和下限显示为刻度或标签;

                                  • palette: 指定一个调色板函数, 用于设置数据可视化中的颜色映射;

                                  • right: 指定分箱间隔是否采用左开右闭, 否则分箱间隔将采用左闭右开;

                                  • rescaler: 指定一个函数, 用于将输入的数值范围缩放到 [0, 1] 的范围内;

                                  • n.breaks: 指定轴或图例显示主要刻度的数量, 在没有设置breaks时生效;

                                  • ...: datetime_scale(···)中要传递给continuous_scale(···)的参数;

                                  • limits: 指定标度的范围, 超出这个范围的数据将由oob指定的函数进行处理;

                                  • timezone: 指定日期时间对应的时区, 可以通过函数OlsonNames查看支持的时区;

                                  • guide: 指定标度美学指南的类型或由guide_Xxx(···)设置美学指南的相关内容;

                                  • position: 指定标度的显示位置(仅适用于坐标轴), 可选"top"、"left"、"right"、"bottom";

                                  • na.value: 指定一个值用于替换缺失值NA(连续型数据), 或指定失值NA要显示的美学状态(离散型数据);

                                  • aesthetics: 指定美学名称, 如"fill"、"size"等, 可以同时指定多个美学, 若c("fill", "colour");

                                  • date_labels: 指定日期时间标签格式规范的字符串, 如"%Y-%m-%d"、"%H:%M:%S"等, 其优先级别高于labels;

                                  • date_breaks: 指定日期时间标度主要刻度的显示间隔, 如"2 weeks"、"3 years"等, 其优先级别高于breaks;

                                  • nice.breaks: 指定是否由程序识别并设置合适的轴或图例显示刻度或标签的位置, 若设置了breaks则忽略该参数;

                                  • date_minor_breaks: 指定日期时间标度次要要刻度的显示间隔, 如"2 weeks"、"3 years"等, 其优先级别高于minor_breaks;

                                  • expand: 指定标度的扩充范围, 由expansion(···)进行设置, 默认对于连续变量上下各扩充5%, 对于离散变量上下各扩充0.6个单位;

                                  • oob: 指定一个函数, 用于处理超出limits范围的数据点, scales::censor将其替换为NA, scales::squish将其压缩到limits范围内(不包含无穷值), scales::squish_infinite将其压缩到limits范围内(包含无穷值);

                                  标度函数

                                  • ...: 传递给构造器的其他参数;

                                  • solid: 指定形状(shape)是否为实心图形;

                                  • alpha: 指定颜色的不透明度, 范围[0, 1];

                                  • colo(u)rs: 指定多色渐变所使用的颜色向量;

                                  • max_size: 指定美学元素(点或文本)的最大尺寸;

                                  • midpoint: 指定两级渐变中间过度点对应的数据值;

                                  • low / mid / high: 指定渐变颜色条上、中、下的颜色;

                                  • h.start: 指定色相轮调色板的起始色调, 范围[0, 360];

                                  • start / end: 指定灰度调色板的起始灰度值和结束灰度值;

                                  • begin / end: 指定颜色映射的起始和结束色调, 范围[0, 1];

                                  • space: 指定计算颜色渐变时的颜色空间, 目前只能使用"Lab";

                                  • range: 指定美学状态的范围, 对于alpha必须是[0, 1]之间的值;

                                  • sec.axis: 指定辅助(副)轴, 由sec_axis(···)或dup_axis(···)进行设置;

                                  • direction: 指定调色板中颜色呈现顺序, 1表示正序或顺时针, -1表示倒序或逆时针;

                                  • h / c / l: 指定色相轮调色板所使用的色调范围[0, 360]、饱和度、亮度[0, 100];

                                  • type: 指定美学元素(颜色或填充)的映射方式:

                                    • 对于scale_fill/colo(u)r_discrete(···)用于指定调色板包含的颜色向量或列表;

                                    • 对于scale_fill/colo(u)r_binned/continuous(···)用于指定调色板的渐变类型(常规"gradient"、色盲"viridis");

                                    • 对于scale_fill/colo(u)r_distiller/brewer/fermenter指定调色板类型(单渐变"seq"、离散"qual"、两极渐变"div");

                                  • values: 指定离散型数据与美学状态之间的映射关系, 若提供一个非命名向量, 则按顺序进行匹配, 若提供一个命名向量, 将会基于名称进行匹配;

                                  • palette: 指定要使用的RColorBrewer调色板, 若为设置type, 可以指定调色板的名称, 若设置了type, 则可以通过索引指定从对应类型调色板列表中进行选择;

                                  • option: 指定颜色的映射模式, 可选"magma" / "A"、"nferno" / "B"、"plasma" / "C"、"viridis" / "D"、"cividis" / "E"、"rocket" / "F"、"mako" / "G"、"turbo" / "H";

                                  延迟美学

                                  理解
                                  xxxxxxxxxx
                                  图层会对其接收的原始数据进行统计变换并产生新的变量, 复合标度会建立美学状态和原始数据间的映射关系, 所谓"延迟美学"就是ggplot提供了一些方法, 可以让用户在"aes(···)"中去访问并应用统计变换产生的新变量以及复合标度建立的美学映射关系;

                                  方法
                                  参数描述

                                  • start: 图层接接收的原始数据中的变量(表达式);

                                  • after_stat / after_scale: 由统计变换(Stat)产生的新变量(表达式)或由复合标度(Scale)换后的美学元素(表达式);

                                  • x: 由统计变换(Stat)产生的新变量(表达式)或由复合标度(Scale)换后的美学元素(表达式); 对于after_stat(···)，可通过统计变换板块的Computed Variables信息查找可用的计算变量;

                                  应用方式:
                                  xxxxxxxxxx
                                  aes(···, 美学元素 = stage(···), ···)
                                  aes(···, 美学元素 = after_stat(···), ···)
                                  aes(···, 美学元素 = after_scale(···), ···)

                                  美学指南

                                  参数描述
                                  导轨

                                  • axis: 指定是否沿着指南(图例)导轨绘制小轴;

                                  • axis.colour: 指定指南(图例)导轨小轴的颜色;

                                  • axis.linewidth: 指定指南(图例)导轨小轴的线宽;

                                  • axis.arrow: 指定指南(图例)导轨小轴的末端的箭头, 由arrow(···)进行设置;

                                  标题

                                  • title.hjust: 指定指南(图例)标题的水平对齐锚点;

                                  • title.vjust: 指定指南(图例)标题的垂直对齐锚点;

                                  • title: 指定指南(轴/图例)标题的字符串或表达式, 如果为NULL, 则不显示标题;

                                  • title.theme: 指定南(图例)标题的主题属性, 由element_text(···)进行设置;

                                  • title.position: 指定指南(图例)标题的位置, 可选"top"、"left"、"right"、"bottom";

                                  标签

                                  • label: 指定是否绘制指南(图例)的文本标签;

                                  • label.hjust: 指定指南(图例)文本标签的水平对齐锚点;

                                  • label.vjust: 指定指南(图例)文本标签的垂直对齐锚点;

                                  • check.overlap: 指定是否删除指南(轴)重叠的文本标签;

                                  • n.dodge: 指定指南(轴)标签交错显示的行数(横轴)或列数(纵轴);

                                  • label.theme: 指定南(图例)文本标签的主题属性, 由element_text(···)进行设置;

                                  • label.position: 指定指南(图例)文本标签的位置, 可选"top"、"left"、"right"、"bottom";

                                  • angle: 指定指南(轴)文本要旋转的角度, 区别于element_text(···)，这里将根据角度自动设置hjust和vjust;

                                  键或颜色条

                                  • nrow / ncol: 指定指南(图例)键的行数列数;

                                  • frame.colour: 指定指南(图例)颜色条边框的颜色;

                                  • ticks: 指定是否在指南(图例)颜色条上绘制刻度线;

                                  • byrow: 指定指南(图例)键的排列方式是否按行排列;

                                  • ticks.colour: 指定指南(图例)颜色条刻度线的颜色;

                                  • frame.linetype: 指定指南(图例)颜色条边框的线型;

                                  • draw.ulim: 是否绘制指南(图例)颜色条的上限刻度线;

                                  • draw.llim: 是否绘制指南(图例)颜色条的下限刻度线;

                                  • frame.linewidth: 指定指南(图例)颜色条边框的线宽;

                                  • ticks.linewidth: 指定指南(图例)颜色条刻度线的线宽;

                                  • nbin: 指定指南(图例)颜色条的容器数量, 值越大, 颜色条越平滑;

                                  • keywidth: 指定指南(图例)键的宽度, 一个数值或由unit(···)进行设置;

                                  • keyheight: 指定指南(图例)键的高度, 一个数值或由unit(···)进行设置;

                                  • barwidth: 指定指南(图例)颜色条的宽度, 一个数值或由unit(···)进行设置;

                                  • barheight: 指定指南(图例)颜色条的高度, 一个数值或由unit(···)进行设置;

                                  • raster: 指定是否将指南(图例)颜色条渲染为光栅图像(并非所有图形设备都能实现该功能);

                                  • default.unit: 指定指南(图例)键或颜色调宽高的默认unit(···)单位, 当keywidth、keyheight、barwidth或barheight为数值时使用该默认单位;

                                  其他元素

                                  • reverse: 指定是否将指南(图例)倒序绘制;

                                  • override.aes: 指定指南(图例)的美学参数列表(list);

                                  • show.limits: 指定是否显指南(图例)范围的上限和下限;

                                  • direction: 指定指南(图例)的绘制方向, 可选"horizontal"、"vertical";

                                  • even.steps: 指定指南(图例)颜色条的容器箱的渲染大小是否相等, 否则将与其多占的数据空间成比例;

                                  应用方式
                                  xxxxxxxxxx
                                  # 法一:
                                  通过"scale_Xxx_Xxx(guide = guide_Xxx(···))"进行设置;
                                  xxxxxxxxxx
                                  # 法二:
                                  先通过"scale_xxx_xxx(guide = 'bins/none/legend/colourbar/coloursteps')"指定"Guide"类型; 之后通过"guides(fill/color/shape/size/··· = guide_Xxx(···))"进行设置;

                                  坐标系统

                                  参数描述

                                  • default: 指定是否将其设置为默认坐标系;

                                  • theta: 指定极坐标要映射的轴"x"或"y";

                                  • x / y: 指定要对横纵坐标轴使用的转换器;

                                  • xlim / ylim: 指定横纵坐标轴要显示的范围;

                                  • ratio: 指定固定纵横比的笛卡尔坐标系的纵横比;

                                  • start: 指定极坐标绘图的起始角度, 一个弧度制角度;

                                  • expand: 指定是否对轴进行扩展, 以确保数据和轴不重叠;

                                  • direction: 指定极坐标绘图方向, 1表示顺时针, -1表示逆时针;

                                  • clip: 指定是否应该将绘图裁剪到绘图面板的范围内, "on"开启裁剪功能, "off"关闭裁剪功能;

                                  Tip:
                                  xxxxxxxxxx
                                  "xlim(···)/ylim(···)"及"scale_x/y_continuous(···)"中参数"limits = "针对的是数据, 先对数据进行筛选, 再使用筛选后的数据绘图;
                                  xxxxxxxxxx
                                  "coord_cartesian(···)"中的参数"xlim/ylim = "与"expand_limits(···)"针对的是坐标系, 先使用原始数据绘图, 再对坐标系进行截取缩放;

                                  Theme
                                  xxxxxxxxxx
                                  library(ggpubr)
                                  Themes <- sprintf('%s()', ls("package:ggplot2", pattern="^theme_"))
                                  Themes <- Themes[! Themes %in% c("theme_set()", "theme_void()")]
                                  Theme.Plots <- lapply(Themes, function(Theme){
                                    return(
                                      ggplot(data.frame(X = 0, Y = 0, Theme = Theme) ,aes(x = X, y = Y)) + 
                                        geom_point(size=3,shape=21,fill='black') + 
                                        scale_x_continuous(limits = c(-1, 1), breaks =  seq(from = -1, to = 1, by = 0.5)) + 
                                        scale_y_continuous(limits = c(-1, 1), breaks =  seq(from = -1, to = 1, by = 0.5)) +

                                  位置调整

                                  参数描述

                                  • seed: 为扰动处理设置随机种子;

                                  • height: 指定垂直方向的扰动量;

                                  • padding: 指定不同组分之间填充比例;

                                  • x / y: 水平方向或垂直方向上移动的距离;

                                  • reverse: 指定是否翻转排列或堆叠的顺序;

                                  • vjust: 指定垂直堆叠的对齐锚点, 适用于点、线、文字;

                                  • dodge.width: 同position_dodge(···)的width参数;

                                  • jitter.width: 同position_jitter(···)的width参数;

                                  • jitter.height: 同position_jitter(···)的height参数;

                                  • width: 对于position_dodge(···)用于指定不同组分之间的填充比例, 对于position_jitter(···)用于指定水平方向的扰动量;

                                  • preserve: 指定对于缺失组分的位置处理方式，"total"将缺失分组所占的空间分配给其他分组, "single"将保留缺失分组所占据的空间;

                                  应用方式
                                  xxxxxxxxxx
                                  通过"geom_Xxx(position = position_Xxx(···))"进行设置;

                                  分面系统

                                  xxxxxxxxxx
                                  library(ggplot2)
                                  Wrap.Plot <- ggplot(data.frame(X = 0, Y = 0, Group = LETTERS[1:4]), aes(x = X, y = Y)) +
                                    geom_text(label = "···") +
                                    scale_x_continuous(name = NULL, breaks = NULL) +
                                    scale_y_continuous(name = NULL, breaks = NULL) + 
                                    theme_test() +
                                    coord_fixed() +
                                    facet_wrap(vars(Group))
                                  Grid.Plot <- ggplot(data.frame(X = 0, Y = 0, Group1 = rep(c("X", "Y"), each = 2), Group2 = rep(c("A", "B"), times = 2)), aes(x = X, y = Y)) + 
                                    geom_text(label = "···") + 
                                    scale_x_continuous(name = NULL, breaks = NULL) + 
                                    scale_y_continuous(name = NULL, breaks = NULL) + 
                                    theme_test() + 
                                    coord_fixed() + 
                                    facet_grid(rows = vars(Group1), cols = vars(Group2))
                                  参数描述

                                  • nrow / ncol: 指定分面行数或列数;

                                  • dir: 指定分面的排列方向, "h"水平排列, "v"垂直排列;

                                  • shrink: 指定各分面尺寸是否适应统计变换后的数据，而不是原始数据;

                                  • drop: 指定是否删除数据中未使用的分面因子, 即不显示没有数据的分面板块;

                                  • margins: 指定是否添加边际子图, 或指定要创建边际图的变量集(存在于分面变量);

                                  • labeller: 指定一个用于分面标签格式化的Labeller函数, 默认是"label_value";

                                  • strip.position: 指定分面标签的显示位置, 可选"top"、"left"、"right"、"bottom", 优先级低于switch参数;

                                  • facets: 指定分面参考变量集, 可通过vars(···)进行设置, 亦可通过形如~Xxx + Xxx + ···的单侧formula进行设置;

                                  • as.table: 指定是否使分面的布局类似于表格，即使得因子级别最高的板块位于右下角，否则因子级别最高的板块位于右上角;

                                  • scales: 指定各分面的坐标系的显示模式, "fixed"所有面板统一, "free"各面板横纵坐标可自由变换, "free_x"或free_y各面板仅横或纵坐标可自由变换;

                                  • space: 指定各分面尺寸的显示模式, "fixed"所有面板尺寸统一, "free"各面板横纵尺寸按坐标轴比例自由变换, "free_x"或"free_y"各面板仅横或纵尺寸可按比例自由变换;

                                  • rows / cols: 指定横纵分面参考变量集, 可通过vars(···)进行设置, 亦可通过形如.~Xxx + Xxx + ···(纵)、··· + Xxx + Xxx~.(横)、··· + Xxx + Xxx~Xxx + Xxx + ···(纵横)的双侧formula进行设置;

                                  • switch: 控制分面标签的显示位置，默认纵向分面标签显示在上, 方横向分面标签显示在右侧, "x"纵向分面标签显示在底部, "y"横向分面标签显示在左侧, "both"同时设置"x"和"y", 优先级高于于strip.position参数；

                                  对于"element_Xxx(···)"中可以通过数值指定尺寸的参数(如size、linewidth、lineheight等), 可以通过"rel(x)"函数指定子级元素相对于父级元素的尺寸;

                                  子母结构

                                  参数描述

                                  • angle: 指定绘图视口的旋转角度(逆时针);

                                  • name: 指定绘图视口的名称，将作为视口的标识;

                                  • gp: 指定绘图视口的基本参数列表, 由gpar(···)进行设置;

                                  • x / y: 指定子视口在父视口中的位置, 一个数值或由unit(···)进行设置;

                                  • layout: 指定绘图视口的布局(几行几列等), 由grid.layout(···)进行设置;

                                  • width / height: 指定绘图视口的宽度和高度, 一个数值或由unit(···)进行设置;

                                  • xscale / yscale: 指定横纵坐标的显示范围, 应用于grid.xaxis(···)和grid.yaxis(···);

                                  • default.units: 指定默认的unit(···)单位, 当x、y、width、height为数值时使用该默认单位;

                                  • just: 指定绘图视口的锚点位置, 可选 "top"、"left"、"right"、"bottom"、"center", 亦可通过一个二元数值向量设置;

                                  • mask: 指定是启用蒙版功能能, “none” / FALSE关闭蒙版功能, “inherit” / TRUE从父视口中继承蒙版, 亦可传入一个grob对象自定义一个蒙版;

                                  • layout.pos.row / layout.pos.col: 指定当前绘图视口在父视口中的行和列, 父级绘图视口需要设置layout, 应用行列布局后将忽略x、y、width、height的设置;

                                  • clip: 指定是否应该将绘图裁剪到视口或自定义的范围内, "on" / TRUE开启裁剪功能, "off" / NA关闭裁剪功能, “inherit” / FALSE从父视口中继承该属性, 亦可传入一个grob对象自定义一个剪切路径;

                                  应用方式
                                  xxxxxxxxxx
                                  Main.Plot <- ···
                                  Child.Plot <- ···
                                  View.Plot <- viewport(···) # 创建绘图视口

                                  print(Sub.Plot, vp = View.Plot) # 输出子图至目标绘图视口

                                  如何避免标签重叠?
                                  xxxxxxxxxx
                                  需求: 尽管"geom_text(···)"提供了参数"check_overlap"用于隐藏重叠的文本标签, 但很多时候我们不希望这些重叠的标签仅是被简单的隐藏, 而是期待它们可以在尽量避免重叠的情况下被绘制出来;
                                  xxxxxxxxxx
                                  方法: 对于这一需求, 可以使用"ggrepel::geom_text_repel(···)"和"ggrepel::geom_label_repel(···)"来实现, 这两个函数利用力学排斥原理, 可以在绘制文本标签的同时尽量避免标签重叠;
